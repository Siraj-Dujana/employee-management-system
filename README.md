# Employee Management System (MERN Stack)

A full-stack Employee Management System built using the MERN stack.  
This project demonstrates CRUD operations with a clean backend API and a React frontend interface.

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

## Features

- Add Employee
- View Employees
- Update Employee
- Delete Employee
- REST API with Express
- MongoDB database integration

## Project Structure

backend/
- Express server
- MongoDB connection
- Employee model
- CRUD routes

frontend/
- React interface
- Axios API calls
- Employee table
- Add/Edit employee forms

## API Routes

GET /api/employees  
POST /api/employees  
PUT /api/employees/:id  
DELETE /api/employees/:id

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend


cd frontend
npm install
npm run dev
